# EdTech React Assessment API

## Base URL
After deploying to Vercel, your base URL will be:
https://your-project-name.vercel.app

## Endpoints

### GET All Courses
GET /api/courses

Headers:
x-api-key: EDTECH2026

### GET Single Course
GET /api/course?id=1

Headers:
x-api-key: EDTECH2026

## Deployment Steps

1. Upload this folder to GitHub
2. Go to https://vercel.com
3. Import the repository
4. Deploy
5. Share the deployed URL with students

This API is:
- 100% Free
- Serverless
- Safe for 300 students
- No database required
